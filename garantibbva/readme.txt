GarantiBBVA payment gateway  integration is a secure and easily integrated payment solution developed for WooCommerce.

== Description ==

GarantiBBVA payment gateway  integration is a comprehensive payment solution designed for e-commerce sites using WooCommerce. It provides your customers with a seamless payment experience through installment options and a modern interface.

Features:

* Easy integration
* Installment payment options
* Modern and classic theme options
* Secure payment infrastructure
* Detailed reporting
* Customizable payment page
* Responsive design

Supported Cards:

* Bonus Card
* Miles&Smiles

== Data Processing and Sharing Policy ==

GarantiBBVA Payment Gateway integrates with payment processing API(s) of authorised Banks and Payment Service Providers (PSP) to accomplish the secure payment processing for your WooCommerce store. The following data can be shared with the Bank or the PSP according to the payment process requirements of the service.

= What Data is Shared with PSPs/Banks =

1. **Transaction Data**
   * Order Information: Order ID, cart total, currency
   * Cart Items: Product names, prices, quantities
   * Discount Information: Applied coupon codes and amounts
   * Shipping Information: Shipping costs and method

2. **Customer (buyer) Data (fraud protection)**
   * Basic Contact Information: Customer's first name, last name, email address, and phone number
   * Billing Address: Street address, city, state, postal code, and country
   * Shipping Address: Shipping recipient's name and address details
   * IP Address: Customer's IP address for fraud prevention and security

3. **Merchant/Entity Information**
   * Store Information: Store name, URL, store/admin email, phone, and address
   * Payment Settings: Currency and payment configuration, payment and refund policy

= When Data is Shared with a PSP/Bank/Partner =

Data can be shared at specific events in the payment process:

1. During Payment Initialization: When a customer proceeds to checkout and selects the GarantiBBVA payment method
2. During Payment Processing: When the customer enters their payment details in the secure iframe
3. During Payment Validation: After the payment is processed, to confirm the payment status
4. During Administrative Operations: When store administrators configure the payment gateway

= Why This Data Sharing is Necessary =

The sharing of this data with PSPs/Banks is essential for the following reasons:

1. **Secure Payment Processing**: Our payment processing partners adopt compatibility with international and industrial secure payment infrastructure standards
2. **Fraud Prevention and Protection**: Customer data helps detect and prevent fraudulent transactions
3. **Installment Options**: To provide installment payment options with appropriate plans and rates
4. **Transaction Verification**: Complete order details are necessary to verify and validate transactions
5. **Regulatory Compliance**: To meet regulatory requirements for financial transactions
6. **Customer Experience**: Enabling a seamless checkout experience with personalized payment options

= Data Security and Privacy =

GarantiBBVA implements several security measures to protect the shared data:

1. Secure API Communication with HTTPS and authentication
2. Tokenization of payment information
3. Minimal data collection - only necessary data is shared
4. Secure iFrame Technology ensuring sensitive payment information never passes through your website's servers
5. Compliance with relevant data protection regulations


== Installation ==

1. Upload the plugin to your WordPress plugin directory
2. Activate the plugin from WordPress admin panel
3. Configure GarantiBBVA from WooCommerce > Settings > Payments section
4. Enter your username and password to log in, add your payment gateway  information from your bank
5. Start accepting payments!

== Frequently Asked Questions ==

= How can I customize installment options? =

You can set installment options and commission rates from the GarantiBBVA management panel.

= Can I change the installment theme? =

Yes, we offer two different theme options: modern and classic. You can select your preferred theme from WooCommerce settings.
